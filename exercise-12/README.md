# Intro to CSS

## Purpose:
Gentle Introduction to CSS

## Learning Objectives:

- Automators will be able to understand the syntax of a CSS rule
- Automators will be able to utilize CSS with Selenium
- Automators will be able to create CSS selectors:
`$selector_id {
  property : value;
}`
